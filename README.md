### What I'm Working On

**CIRCE** — Multi-agent AI orchestration framework running on a Jetson Orin Nano 8GB. Nine specialized agents (backend, frontend, database, devops, AI/ML, security, testing, automation, docs) coordinated by a master orchestrator, communicating through file-based protocols with full human oversight.

- Persistent agent memory across sessions
- File-based inter-agent communication (no message bus required)
- Dual-environment: Jetson edge + GCP cloud VM
- Voice pipeline: ElevenLabs TTS + GCP Speech-to-Text
- Production stack: Next.js, FastAPI, PostgreSQL+pgvector, Docker
